## Simple Bank

Nutrifood Test - Console App

### Installation steps:

1. Clone project from https://github.com/imkhoirularifin/nutrifood-test.git

```shell
git clone https://github.com/imkhoirularifin/nutrifood-test.git
```

2. Navigate to project directory

```shell
cd nutrifood-test
```

3. Run the project

```shell
dotnet run
```
